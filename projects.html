FireWatch @ HackUCI - February 2019

This web app uses machine learning and over 400,000 data points to predict where wildfires are most likely to occur in California. Python and Flask used to parse data, train an algorithm, and host a web server.

RGear Card Swiper - February 2019

This application was used by the Associated Students of UC Riverside (ASUCR) for the sponsorship of free sweaters for undergratuates to promote campus pride. This application increased the throughput of the distribution of sweaters from 4 hours to 2.5 hours (30% speedup).

iSpy Web Game @ Citrus Hack - April 2018

A web-game constructed in 24 hours at UCR's Citrus Hack, using HTML, CSS and JS all for the first time.

Bulletin Board C++ Program - March 2018

A C++ program that allows users to create an account, post messages to a public page, and access past messages. Messages and user login information and loaded in from local files.
